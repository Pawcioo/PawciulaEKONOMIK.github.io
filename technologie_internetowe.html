Technologie internetowe

Języki tworzenia stron internetowych

HTML - (HyperText Markup Language)

Język HTML odpowiada za strukturę i zawartość strony internetowej. Obecnie stosuje się HTML w wersji 5

CSS - (Cascading Style Sheets

Język CSS odpowiada za wygląd strony internetowej. Obecnie stosuje się CSS w wersji 3.

Języki programowania

Język JavaScript

Język JavaScript działa po stronie klienta (w przeglądarce) i zapewnia dodatkowe funkcjonalności strony internetowej, np. automatyczne sprawdzanie formularzy przed wysłaniem.

Język PHP

Język PHP działa po stronie serwera i umożliwia np. połączenie z bazą danych.

Język SQL

Język SQL nie jest właściwie językiem programowania, lecz językiem obsługi baz danych. Działa po stronie serwera stron internetowych w połączeniu z językiem PHP.
